# goops-monitor

A robust monitoring tool for Goops.

## Overview

`goops-monitor` provides real-time tracking and analytics for Goops services. It consists of a monitor service and a runner for executing checks.

## Features

- Real-time monitoring
- Automated runner execution
- Extensible tutorial system

## Installation

```bash
go get github.com/Maximus-08/goops-monitor
```

## Usage

```bash
go run monitor/main.go
```
